|:gear:| Installation
===========================

It is recommended to install MOXελ inside a virtual environment.

1. Create a virtual enviroment and activate it:

   .. code-block::

        $ python -m venv <venvir_name>
        $ source <venvir_name>/bin/activate

2. Install ΜΟΧελ:

    .. code-block::

        (<venvir_name>) $ pip install moxel
